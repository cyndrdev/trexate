# Trexate

A time-travelling twinstick shooter, built for a 12-week university module. Take charge against waves of enemies as you fight your way back in time to save the galaxy!

## Controls
### Keyboard & Mouse

- WASD to move
- Mouse to aim, click to shoot
- ESC to pause

### Controller

- Left stick to move
- Right stick to shoot
- START to pause
